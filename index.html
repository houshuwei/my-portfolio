📁 Projects

🎮 Grid World Treasure Hunt with Q-learning
Sep 2025 – Dec 2025 | Individual Project
📌 Problem: An agent must navigate an 8×8 grid to collect all treasures while avoiding traps, learning optimal behavior via trial and error.
📁 Data: Synthetic grid environment (8×8, 4 treasures, multiple traps). State = position + treasure bitmask (1024 states). Actions: up/down/left/right.
⚙️ Approach: Tabular Q-learning with ε-greedy exploration (ε decays from 1.0 to 0.01). Reward: +10 per new treasure, -5 for trap, -0.1 per step. Trained for 2500 episodes.
📈 Outcome: Agent collects all 4 treasures in ~35 steps (near-optimal). Average reward stabilizes around 34 (max possible ~40). Learned policy avoids all traps.
🙋 Contribution: Full implementation in Python (NumPy, Matplotlib), designed state encoding, reward function, training loop, and real-time visualization.

💳 Credit Card Fraud Detection (Imbalanced Classification)
Jan 2026 – Mar 2026 | Group Project (6 members)
📌 Problem: Identify fraudulent transactions from highly imbalanced credit card data (fraud rate only 0.17%) to minimize financial loss and false positives.
📁 Data: Kaggle MLG‑ULB dataset (284,807 transactions, 30 anonymized PCA features + Time & Amount). 80/20 stratified split.
⚙️ Approach: Preprocessing: scaling (Time, Amount) + SMOTE on training set only. Models: Logistic Regression vs. Random Forest (class_weight='balanced'). Evaluation: ROC‑AUC, AUPRC, precision/recall/F1.
📈 Outcome: Random Forest achieved precision 0.911, recall 0.758, F1 0.828, AUPRC 0.812 – far superior to Logistic Regression. Showed that ROC‑AUC can be misleading under extreme imbalance.
🙋 Contribution: Organized the entire codebase, compiled performance metrics and figures, uploaded all materials to GitHub, and prepared the repository README documentation.
