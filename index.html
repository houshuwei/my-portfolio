📁 Projects

📈 Fine-tuning LLM for Financial Sentiment Analysis
Sep 2026 – May 2027 | Individual Project
📌 Problem: Financial firms need real-time sentiment signals from earnings calls and news, but general LLMs lack domain-specific accuracy.
📁 Data: 50k annotated financial news headlines + 10k earnings call transcripts (public datasets: FiQA, FPBs).
⚙️ Approach: LoRA fine-tuning of Llama 2-7B on A100 GPU; optimized prompt templates and applied 4-bit quantization for inference.
📈 Outcome: Achieved 86% F1-score (vs 72% base model) and reduced inference latency by 40% via quantization; deployed a demo API.
🙋 Contribution: Sole researcher – data preprocessing, fine-tuning pipeline, evaluation, and API deployment.

⚡ Efficient LLM Inference for Customer Service Bots
Feb 2027 – Dec 2027 | Group Project (3 members)
📌 Problem: A commercial chatbot powered by LLM had high latency (>3s per query) and could not handle peak traffic cost-effectively.
📁 Data: 200k real anonymized customer queries (e-commerce returns, product info).
⚙️ Approach: Implemented speculative decoding and dynamic batch inference; pruned less important attention heads using SparseGPT.
📈 Outcome: Reduced average latency to 0.9s and cut GPU cost by 55% while maintaining 97% response quality.
🙋 Contribution: Designed speculative decoding algorithm and led integration into production prototype; co-authored technical report.
